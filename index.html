Indice de masa corporal
Calculadora de IMC

El Índice de Masa Corporal (IMC) es una medida que se utiliza para determinar si una persona tiene un peso saludable en relación con su estatura. Es importante conocer nuestro IMC para asegurarnos de que estamos dentro de un rango saludable y tomar medidas para mejorar nuestra salud en caso de que sea necesario.
La fórmula para calcular el IMC es:

IMC = peso (kg) / altura (m)²

Calculadora de porcentaje de grasa corporal

El porcentaje de grasa corporal es otra medida importante para evaluar la salud. La fórmula para calcularlo es diferente para hombres y mujeres. A continuación, te mostramos las fórmulas para ambos:

Para hombres:

1.20 x IMC + 0.23 x edad - 10.8 x 1 - 5.4

Para mujeres:

1.20 x IMC + 0.23 x edad - 10.8 x 0 - 5.4

Un rango saludable para los hombres es de un 8-19% de grasa corporal, mientras que para las mujeres es de un 19-31%. Sin embargo, es importante recordar que estos son solo valores promedio y que cada persona es diferente.
